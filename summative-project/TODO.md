# 人力资源分析（HR Analytics）项目清单

## 1. 项目目标

- 探索员工满意度、项目数量、工时、部门、薪资等因素与离职之间的关系
- 完成基础可视化与统计分析
- 构建离职预测模型（分类任务）
- 作为 Kaggle 入门项目复现完整数据分析流程
- 作为课程期末作业提交一份含可视化与模型的完整报告

___

# 2. 数据探索（EDA）

### 2.1 数据基本情况

- 字段数量与数据类型统计
- 数值字段分布直方图
- 分类字段分布条形图（部门、薪资等级）
- 判断缺失值（本数据集无缺失）
- 检查异常值（如工时过高/过低）

### 2.2 离职情况分布

- left 字段的分布图（条形图或饼图）
- 不同部门的离职率比较
- 不同工资等级的离职率比较

### 2.3 关键因素关系分析

- 满意度 vs 离职
- 月均工时 vs 离职
- 项目数量 vs 离职
- 工龄 vs 离职
- 升职情况 vs 离职
- 工伤 vs 离职

### 2.4 相关性分析

- 数值特征的相关系数矩阵（heatmap）
- 离职与各数值特征的箱线图/小提琴图

# 3. 特征工程与模型构建

### 3.1 特征预处理

- 数值特征标准化处理或不处理（按课程要求选择）
- 类别特征独热编码（部门、工资等级）
- 划分训练集和测试集（70/30 或 80/20）

### 3.2 模型构建（可多模型比较）

- Logistic Regression（可解释性强）
- Random Forest（常用基准模型）
- XGBoost 或 LightGBM（高性能）
- KNN、SVM（可作为补充或比较）
- 简单神经网络（可选）

### 3.3 模型评估指标

- Accuracy
- Precision / Recall / F1-score
- Confusion Matrix
- ROC 曲线与 AUC 分数

# 4. 特征重要性与模型解释

### 4.1 特征重要性（基于树模型）

- 使用 Random Forest 或 XGBoost 提取特征重要性
- 分析影响离职概率最大的变量

### 4.2 使用 SHAP 进行模型解释（进阶）

- 全局 SHAP（summary plot）
- 单样本 SHAP（force plot）

# 5. 可视化设计（Tableau/PowerBI 或 Python）

### 5.1 推荐绘图

- 部门离职率柱状图
- 满意度直方图与箱线图
- 工时分布图
- 工龄分布图
- 项目数 vs 工时散点图
- 薪资等级 vs 离职热力图
- 相关矩阵 heatmap
- 模型 ROC 曲线
- SHAP summary plot

### 5.2 仪表盘建议（Dashboard）

- 总离职率
- 部门离职率
- 薪资等级构成
- 员工满意度分布
- 月均工时分布
- 高风险离职员工预警

___

# 6. 员工分群分析（可选）

使用 K-Means 或其他聚类算法进行分析：

- 高工时低满意度群体
- 工龄长但无升职群体
- 高满意度高薪资的稳定群体
- 使用 PCA 将聚类结果降维可视化
